# Sound Minimum

Концертный аудиоплеер для живых выступлений и фоновой музыки.

## Features
- Список воспроизведения с drag-drop сортировкой
- Кроссфейд между треками
- Фоновая музыка с настраиваемым поведением
- Multi-device вывод
- Кастомный тёмный UI (GDI+)
- Локализация (en/ru)

## Build
```bash
dotnet build -c Release
```
